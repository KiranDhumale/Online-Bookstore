# Online-Bookstore
This SQL project presents a comprehensive analysis of an Online Bookstore database containing multiple book genres and customer transaction records.
The database consists of three relational tables — Customers, Book Details, and Orders — used to extract meaningful business insights through SQL queries.
Advanced SQL concepts such as JOINs, aggregate functions, GROUP BY, subqueries, filtering clauses, and sorting operations were applied to analyze sales and customer behavior.
The project provides valuable insights into inventory management by identifying out-of-stock books and monitoring product availability.
Customer purchasing trends were analyzed to determine the most popular book genres, top-performing authors based on sales volume, and countries contributing the highest customer base.
The analysis supports data-driven business decisions related to inventory planning, customer targeting, product demand, and sales performance optimization in an online bookstore environment.
